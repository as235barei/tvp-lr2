# TechShop — Лабораторна робота №2

Тема роботи — оформлення сайту інтернет-магазину TechShop за допомогою CSS.
HTML-сторінки з попередньої роботи доповнені власним файлом стилів
`prototype/css/styles.css`, який підключений до всіх сторінок.

## Технології

- HTML5
- CSS3

## Структура

- `prototype/` — сторінки сайту та файл стилів `css/styles.css`
- `diagrams/` — діаграма прецедентів (use-case)

## Як запустити

Відкрити файл `prototype/index.html` у браузері.

Демо: https://techshop-inky.vercel.app
